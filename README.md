# Tutor Website (Tailwind + Static HTML)

## Quick start
1) Install Node.js (LTS)
2) In this folder, run:
   - npm install
   - npm run build:css   (or npm run watch:css)
3) Open `index.html` with the VS Code Live Server extension.

## Edit these first
- `index.html` (homepage content)
- `courses.html` (course cards + links)
- `book-session.html` (Calendly iframe URL)

## Where to add your media
- Replace placeholder images in `/images`
- Swap the sample YouTube embed with your own (or Vimeo)
